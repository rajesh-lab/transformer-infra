# transformer-infra
contains  🏗️ infra commonly required to work with transformer and friends.

We use this internally for research purposes. 

Instead of functioning as a library, it's used more like a common-place to have useful code one can build off custom repos (copy-paste from) and try research ideas fast. As a result, most implementations are minimal, single-file and importantly, wherever possible, as fast as possible to not compromise with speed of iteration.

Let us know if you need any feature developed ⚡️ (or report any bugs 🐞).
